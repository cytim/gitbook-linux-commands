# Elixir - IEx

The Interactive Elixir shell. It supports most of the elixir arguments.

**Execute Script**

```bash
iex -S <SCRIPT>
```

**Require File**

```bash
iex -r <FILE>
```

**Recompile and Reload the Module**

```
iex> r <MODULE>
```
